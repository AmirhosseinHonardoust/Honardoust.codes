EXP-003 / question

Technical question

Which synthetic data approach better preserves the statistical behavior of a real tabular dataset?

EXP-003 / method

Method and workflow

  1. Train Gaussian Copula and VAE generators on the same tabular dataset.
  2. Generate comparable synthetic datasets.
  3. Evaluate fidelity using distribution overlap and correlation difference.
  4. Use PCA projection and pairplots for visual comparison.
  5. Produce automated visual reports.

EXP-003 / evidence

Evidence of work

Evaluation metrics

The README defines distribution overlap, correlation difference, PCA projection, and pairplot comparison.

Quantitative result

The comparison reports Copula with higher mean distribution overlap and lower mean correlation difference than VAE.

Reproducibility

The project includes config, reports, outputs, and separate generated synthetic datasets.

EXP-003 / limitations

Limitations and honesty check

  • Synthetic data quality depends strongly on dataset type, sample size, and feature distributions.
  • Privacy leakage testing is not the same as fidelity testing and would need separate analysis.
  • Utility should also be tested downstream by training models on synthetic data and testing on real data.
