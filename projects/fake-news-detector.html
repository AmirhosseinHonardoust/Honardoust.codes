EXP-004 / question

Technical question

How far can a clean classical NLP pipeline go for fake-vs-real news classification?

EXP-004 / method

Method and workflow

  1. Load balanced real and fake datasets.
  2. Clean and merge text data.
  3. Extract TF-IDF features.
  4. Train Logistic Regression.
  5. Save model artifacts, vectorizer, metrics, charts, and a Streamlit app.

Evidence of work

Dataset

The README describes 999 real and 999 fake rows from an adapted Kaggle Fake and Real News dataset.

Pipeline

The training script loads datasets, cleans data, extracts TF-IDF features, trains Logistic Regression, and saves artifacts.

Interface

EXP-004 / limitations

Limitations and honesty check

  • Perfect accuracy on a cleaned dataset can signal dataset simplicity or leakage risk; out-of-domain evaluation is essential.
  • Real fake-news detection requires source credibility, temporal drift, adversarial robustness, and multilingual coverage.
  • Classifier outputs should be communicated as signals, not absolute truth.
